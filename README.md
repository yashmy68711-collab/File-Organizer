# File Organizer (Python)

## About

This project is a Python script that automatically organizes files in a folder based on their file types.
It helps keep folders clean by sorting files into categories like Images, Documents, Videos, and more.

## Features

* Automatically detects file types
* Creates folders if they don’t exist
* Moves files into appropriate folders
* Supports multiple formats (jpg, png, pdf, mp4, zip, etc.)
* Shows log messages for each moved file

## 🛠️ Tech Used

* Python
* os module
* shutil module

## How to Run

1. Run the Python file
2. Enter the folder path when asked
3. Files will be automatically organized
